# Discord KI Überwachungs-Bot

## Features
- Slash Command `/status`
- Steuerung per Buttons (DM only)
- 3 Modi:
  - NORMAL
  - ALARM (Überwachung)
  - ALARM_NOW (sofort)

## Hosting
- Render (Web Service)
- Secrets über Environment Variables

## Environment Variables
DISCORD_TOKEN=dein_bot_token  
OWNER_ID=deine_discord_id  

## Start
```bash
python bot.py
